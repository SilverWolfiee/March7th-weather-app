# March7th Themed Weather App

This is a March7th Themed weather widget app that i code using HTML, CSS and JS with Tauri as the app backend.


## GET STARTED

1. git clone this repository

2. run ```npm install```

3. You need a [backend server](https://github.com/SilverWolfiee/elysia-weather) to get the weather data. Point it in .env
(do note that my backend server uses bun and elysiajs, if you're on windows you might need to use WSL)

4. Run ```npm run tauri dev``` to enter dev mode

## RELEASE BINARIES

### For Windows
Navigate to release and download the setup installer

### For Linux
Clone this repository then run ```npm run tauri build```
